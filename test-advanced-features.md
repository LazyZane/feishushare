## 3. Mermaid 图表测试

### 流程图

```mermaid
graph TD
    A[开始] --> B{是否登录?}
    B -->|是| C[显示主页]
    B -->|否| D[显示登录页]
    D --> E[用户登录]
    E --> F{登录成功?}
    F -->|是| C
    F -->|否| G[显示错误信息]
    G --> D
    C --> H[结束]
```

### 序列图

```mermaid
sequenceDiagram
    participant U as 用户
    participant C as 客户端
    participant S as 服务器
    participant D as 数据库

    U->>C: 发起请求
    C->>S: 转发请求
    S->>D: 查询数据
    D-->>S: 返回结果
    S-->>C: 响应数据
    C-->>U: 显示结果
```
